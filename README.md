# `VUE-WE-CHAT`
## 仿微信网页聊天，使用vue框架，websocket实现聊天文件发送等功能，webRTC实现语音视频聊天等功能

## 前端 client
- 安装依赖 `npm install`
- 本地开发 `npm run dev`
- 打包 `npm run build`

## 后端 server
- 安装依赖 `npm install`
- 本地开发 `npm run dev`

## mongodb
- 开启`mongodb`服务 `sudo mongod`
